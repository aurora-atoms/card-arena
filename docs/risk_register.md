# Risk Register

| Risk | Impact | Mitigation |
|---|---:|---|
| Invalid actions | High | Use official legal-action mask and tests. |
| Slow decisions | High | Add timeout fallback to heuristic agent. |
| Rule mismatch | High | Keep all official integration in adapters. |
| Overfitting to baseline | Medium | Maintain multiple opponents and seeds. |
| Public IP/copyright issue | High | Avoid restricted assets and private data. |
| RL instability | Medium | Require deterministic league before training. |
